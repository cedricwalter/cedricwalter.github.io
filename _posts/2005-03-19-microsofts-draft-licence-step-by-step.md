**Microsoft is guilty of breaking EU competition law, and has come up with a way to make amends. We dissect the resulting licence and find a very peculiar world indeed**   
   
*In last year’s European Commission antitrust ruling against Microsoft, the software giant agreed to create a server interoperability licence that would allow rival makers of server software to write applications that can "achieve full interoperability" with Windows client and server operating systems on "reasonable and non-discriminatory terms".* [ZDNet UK Insight](http://insight.zdnet.co.uk/software/windows/0,39020478,39191959,00.htm)
