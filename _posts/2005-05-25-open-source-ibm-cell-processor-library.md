IBM want to open it’s development kit for the famous 8 cores CELL processor (due in PS3 next year and expected in high
end server station). Specs should be released before end of months, and development kit this fall.

The [Cell processor](http://www-1.ibm.com/press/PressServletForm.wss?MenuChoice=pressreleases&TemplateName=ShowPressReleaseTemplate&SelectString=t1.docunid=7502&TableName=DataheadApplicationClass&SESSIONKEY=any&WindowTitle=Press+Release&STATUS=publish)
consists of a general-purpose POWERPC processor core connected to eight special-purpose DSP cores.
