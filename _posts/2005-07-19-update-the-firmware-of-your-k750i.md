| ### Install the latest firmware  Go [to the support homepage of Ericsson](http://www.sonyericsson.com/spg.jsp?cc=global&lc=en&ver=4001&template=ps1_1_3_2&zone=ps&lm=ps1_1&pid=10242) (only with IE not firefox):  - Choose automatic mode - Check the prerequisite (a list with checkboxes) - Install the java application - Connect he usb cable to the phone<span style="FONT-FAMILY: monospace"> - <span style="FONT-FAMILY: monospace"> Accept the licence and let the java applet connect to the phone and download the latest rom for You.<span style="FONT-FAMILY: monospace"> - <span style="FONT-FAMILY: monospace"> A popup windows:  1\. Ask You to switch the phone off, and disconnect it from the usb, do it   2\. While pressing the button "C" to connect it to the usb cable   BUT Windows complains about the driver, even If you have install the full Ericsson suite   [download the driver here](http://www.sonyericsson.com/downloads/D750_K750_W800_Update_Service_USB_driver.zip): (was not found on the genuine cd)  – after installing the driver, the update should start an be completed in 12 minutes more or less…   – I was the force to remove the battery because my phone was temporarly blocked, after the restart, the phone complete the installation and it’s finish.   – verify that You have the latest ROM version   ![](/assets/images/2005/07/sony-ericsson-k750i.jpg) |
