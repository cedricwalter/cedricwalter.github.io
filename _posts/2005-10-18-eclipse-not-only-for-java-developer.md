 I came across [this article](http://www-128.ibm.com/developerworks/opensource/library/os-rubyeclipse/?ca=dgr-lnxw01Ruby4Eclipse) at [IBM developerworks](http://www-128.ibm.com/developerworks), which explain how to use the RDT: Ruby IDE for the [Eclipse platform](http://www.eclipse.org). I decide also to give You here an overview of all languages supported by [eclipse:](http://www.eclipse.org)

- JDT: <span style="font-weight: normal; font-style: italic;">[The JDT project](http://www.eclipse.org/jdt/) provides the tool plug-ins that implement a #Java IDE supporting the development of any #Java application, including #Eclipse plug-ins. It adds a #Java project nature and #Java perspective to the #Eclipse Workbench as well as a number of views, editors, wizards, builders, and code merging and refactoring tools. The JDT project allows #Eclipse to be a development environment for itself.
- <span style="font-weight: bold;">RDT is an [open source Ruby IDE for the #Eclipse platform](http://rubyeclipse.sourceforge.net/).  
    Features supported are syntax highlighting, on the fly syntax check, graphical outline, Test::Unit view/runner, Ruby application launching, content assist, source formatter, and a Ruby debugger.
- <span style="font-weight: bold;">CDT <span style="font-style: italic;"> (C/C++ Development Tools) Project is working towards providing a fully functional [C and C++ Integrated Development Environment ](http://www.eclipse.org/cdt/)<span style="font-style: italic;">(IDE) for the #Eclipse platform. 
- <span style="font-weight: bold;">Cobol: #Eclipse fundation is hosting one [COBOL CDT project](http://www.eclipse.org/cobol/) but there is also a commercial plugin: t<span style="font-style: italic;">he [ACUCOBOL-GT #Eclipse Plug-in](http://www.acucorp.com/support/downloads/index.php)<span style="font-style: italic;">,   
    
- <span style="font-weight: bold;">PHP: [<span style="font-style: italic;"> PHPeclipse](http://sourceforge.net/projects/phpeclipse/) (open source CPL) PHP parser, debugger, code formatter, outline view, templates. Note also that [ZEND](http://www.zend.com/) has joined the eclipse board, so their flagship product ZEND Studio Pro may have a future as an eclipse plugin.
- <span style="font-weight: bold;">PERL: [EPIC is an opensource Perl IDE for the #Eclipse platform](http://e-p-i-c.sourceforge.net/main.html). Features supported are syntax highlighting, on the fly syntax check, content assist, perldoc support, source formatter, templating support and a Perl debugger. A regular expression plugin and support for the eSpell spellchecker are also available.
- <span style="font-weight: bold;">Python: [PyDev](http://pydev.sourceforge.net/) is a plugin that enables users to use [Eclipse for Python and Jython development](http://pydev.sourceforge.net/). It comes with many goodies such as code completion, syntax highlighting, syntax analysis, refactor, debug and many others.
- <span style="font-weight: bold;">Eiffel: The [Eclipe Eiffel Development Tools (EDT)](http://sourceforge.net/projects/eedt/) are a set of plugins which eventually implement a full-fledged, highly integrated Eiffel development environment based on the #Eclipse Platform
- <span style="font-weight: bold;">Groovy: [The plugin provides a Groovy editor](http://groovy.codehaus.org/Eclipse+Plugin) for editing scripts or classes in Groovy (Groovy is an object-oriented programming language designed for the #Java platform as an alternative to #Java with features from Python, Ruby and Smalltalk. see [WikiPedia)](http://en.wikipedia.org/wiki/Groovy) language. Groovy makes writing scripts and applications for the #Java Virtual Machine fast and easy.
- <span style="font-weight: bold;">Microsoft C#: [Improve C# code editing](http://www.improve-technologies.com/alpha/esharp/) with: Creation of new C# files (with a basic structure) – Syntax highlighting – Content assistant for C# keywords – Auto-indentation – C# code compilation etc Unfortunately no more active development since June 2004

And the list goes on……beside this nearly all minor <span style="font-weight: bold;">scripting languages are supported (jsp, javascript … for ex.) and minor<span style="font-weight: bold;"> markup <span style="font-weight: bold;">language (XML, HTML, JSF…) with the tremendous list of plugins:

- [www.eclipseplugincentral.com](http://www.eclipseplugincentral.com) #Eclipse Plugin Central (EPIC) offers the #Eclipse community and users of IBM WebSphere Studio application development environments (WSAD, WSSD, WSED, etc.) a convenient, information-rich portal that helps software developers find open source and commercial #Eclipse plug-ins, tools, and products that enhance the entire #Java software development life-cycle.
