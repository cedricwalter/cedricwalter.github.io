After [Microsoft Warns of New Security Threat System monitoring programs, called rootkits, may pose a serious danger to your PC](http://www.pcworld.com/news/article/0,aid,119720,00.asp). it is time to see what offering is available to protect our PCs…<span style="font-style: italic;">

A root kit is a set of tools used by an intruder after cracking a computer system. These tools can help the attacker maintain his or her access to the system and use it for malicious purposes. Root kits exist for a variety of operating systems such as #Linux, Solaris, and versions of Microsoft Windows. [\[WikiPedia\]](http://en.wikipedia.org/wiki/Rootkit)

The windows rootkit threat has never been so high as today: [Rootkit creators turn professional](http://www.vnunet.com/vnunet/news/2144149/rootkits-turn-professional)  
All major antivirus software are now starting to provide solutions with more or less sucess:

- [Symantec ](http://securityresponse.symantec.com/avcenter/venc/data/hacktool.rootkit.html)<span style="font-style: italic;">[Hacktool.Rootkit](http://securityresponse.symantec.com/avcenter/venc/data/hacktool.rootkit.html) comprises a set of programs and scripts that work together to allow attackers to break into a system. If Hacktool.Rootkit is detected on a system, it is very likely that an attacker has gained complete control of that system. All files that are detected as Hacktool.Rootkit should be deleted. Infected systems may need to be restored from backups or patched to restore security.
- <span style="font-style: italic;">[Sysinternal](http://www.sysinternals.com) is a company more known for his hacking or developer tool, but they have been the first to give away a [free rootkit revealer and detection program](http://www.sysinternals.com/Utilities/RootkitRevealer.html).
- F-Secure Corp has added rootkit-detection features to its product suite: [F-secure Blacklist](http://www.f-secure.com/blacklight/)
- [<span style="font-style: italic;">Microsoft Strider Ghostbuster](http://www.eweek.com/article2/0,1895,1838294,00.asp) is a future tool from the Giant.

The only problem is that the only solution is to restore your system by using a "non corrupted" os version (the problem is to have enough backup)…
