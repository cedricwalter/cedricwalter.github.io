The National Security Agency (NSA)

NSA/ Central Security Service (NSA/CSS) is a United States government agency responsible for both the collection and analysis of message communications, and for the security of government communications against similar agencies elsewhere. It is a part of the Department of Defense. … ) \[[WikiPedia](http://en.wikipedia.org/wiki/NSA)\]

*has developed and distributed configuration guidance for Microsoft Windows NT and Windows 2000 in the form of configuration guides. These guides are currently being used throughout the government and by numerous entities as a security baseline for their Windows systems.  
To assist our Windows XP user community, NSA has developed security configuration guidance for Windows XP, with the cooperation of other government agencies….* [Microsoft Windows XP Guides](http://www.nsa.gov/snac/downloads_winxp.cfm?MenuID=scg10.3.1.1)
