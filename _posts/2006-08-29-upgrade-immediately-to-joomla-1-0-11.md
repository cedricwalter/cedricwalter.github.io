<span style="font-style: italic;">

All existing #Joomla! users <span style="color: red;">**MUST UPGRADE** to this version, due to several <span style="color: red;">High Level vulnerabilities that **affect ALL Previous versions of #Joomla!** 

1.0.11 contains the following critical security fixes:

- **04 High Level Security Fixes**
- 04 Medium Level Security Fixes
- 18 Low Level security
- *25 General bug fixes*

If you are using ANY previous version of #Joomla!, you need to upgrade to 1.0.11 as soon as possible.

- [1.0.11 Version Information](http://www.joomla.org/content/blogcategory/32/66/)
- [1.0.11 Changelog ](http://www.joomla.org/content/view/1841/78/)
- [1.0.11 Package File MD5 checksums](http://www.joomla.org/content/view/1842/95/)

<span style="font-style: italic;">Project #Joomla! is and has always been fully committed to a <span style="font-weight: bold; font-style: italic;">Security First Principle<span style="font-style: italic;"> and new intiatives have and will continue to be started to reinforce and continue this principle. #Joomla! 1.0.11 highlights a redoubled effort to put Security at the forefront of everyones lexicon.
