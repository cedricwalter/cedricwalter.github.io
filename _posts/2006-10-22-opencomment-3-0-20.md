The code is getting better at each release…but may have encounter some regressions. So Your feedback is welcomed

- <span style="font-weight: bold; color: rgb(51, 204, 0);">NEW: use the project [xajax-joomla.org](http://www.xajax-joomla.com) which let you run multiple XAJAX modules or components (1)
- BUG: encoding problem should be defacto solve as XAJAX now use #Joomla encoding
- <span style="font-weight: bold; color: rgb(51, 204, 0);">NEW: Version able to run on hardened PHP server: Register globall OFF (php.ini) + SAFE\_MODE On (php.ini) + RG\_EMULATION = 0 (Joomla globals.php). Should solve different issues like "You are trying to hack me"
- BUG: User homepage was not saved or displayed
- <span style="font-weight: bold; color: rgb(51, 204, 0);">NEW: Better CSS support (see OpenCommentUpdaterA)
- <span style="font-weight: bold; color: rgb(51, 204, 0);">NEW: code refactored and enhance compatibility during sanitization of inputs (mysql\_real\_escape\_string was not existing on all host) see OpenCommentSanitizer
- BUG: minor translations problem.
- <span style="font-weight: bold; color: rgb(51, 204, 0);">NEW: german translations credits
