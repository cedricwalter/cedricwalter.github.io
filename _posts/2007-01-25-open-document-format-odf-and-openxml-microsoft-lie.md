 The Specification of both format are currently being reviewed, OpenXML is containing a lot of details about Office <span style="text-decoration: line-through;">features bugs (sorry any developer can not call them differently). Extract of the OpenXMl specifications are popping up on the web, one site is trying to bring a view on the problem.

<big>"</big>The Open XML specification comes in 5 different PDF files with an 6 accompanying electronic annexes in excess of 43 megabytes. For comparison the [ODF specification](http://www.oasis-open.org/committees/tc_home.php?wg_abbrev=office#technical) is a single 11 megabyte PDF, with 3 separate XML schemas. The ODF specification weighs in at a mere 722 pages, where as, the largest PDF in the Open XML specification is 5219 pages long.<big>" </big>[read more](http://3monkeyweb.com/3monkeys/2007/01/17/the-open-xml-lie/)

<big>   
 </big>The reason why the document is so looong, is that it will force all developer that want to support OpenXML to reproduce Microsoft Office fully (a program code base being 10 years old so good luck), as a result, [Open XML will be nearly fully read and written only by Microsoft products.](http://www.sutor.com/newsite/blog-open/?p=1145)

 That exactly the contrary we would expect from an open standard

 The Organization for the Advancement of Structured Information Standards (OASIS) approved ODF as a standard in 2005, and it is now supported by IBM, Sun ([OpenOffice](http://www.openoffice.org/)) and Novell. [IBM decide to stand against Microsoft OpenXML format lin december 2006.](http://news.com.com/2061-10795_3-6141703.html)

 Suprisingly (not for me) Microsoft has agreed only to support the [Open XML Translator,](http://sourceforge.net/projects/odf-converter) a project on Sourceforge which currently offers translations ONLY between ODF and Word 2007, in clear text they help user migrate from ODF to their <span style="text-decoration: line-through;">closed open format. Things never change, they want to control Your file format, in order to force users staying with them…It is all about choice, support ODF
