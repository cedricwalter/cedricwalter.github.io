This article shows how you can convert a physical Windows system (XP, 2003, 2000, NT4 SP4+) into a VMware virtual
machine with the free [VMware Converter Starter](http://www.vmware.com/products/converter/). The resulting virtual
machine can be run in the free [VMware Player](http://www.vmware.com/products/player/)
and [VMware Server](http://www.vmware.com/products/server/), and also in VMware Workstation and other VMware products.
Vmware Converter comes in handy if you want to switch to a #Linux desktop, but feel the need to run your old Windows
desktop from time to time. By converting your Windows desktop into a virtual machine, you can run it under VMware
Server/Player, etc. on your #Linux desktop.
