# When #Eclipse meet #Joomla!…from 0 to #Joomla! in 3 minutes**

![](/assets/images/2006/04/eclipse.jpg) ![](/assets/images/2005/10/Joomla-Logo-Horz-Color-Thumbnail.png)

The easiest way to develop component, modules, plugin for #Joomla! using best of the breed open source software.   
In order to speed up development of #Joomla software, I did develop a developer package "start and ready to work". So basically nothing special for any experienced developer, just a gain of time. Less experienced developer will enjoy the package! Just unpack the big ZIP file on any drive, start 2 program and you have a running [Eclipse](http://www.eclipse.org/) and [Joomla](http://www.joomla.org)! environment for developing

[Read more in the WIKI](http://wiki.waltercedric.com/index.php?title=SolarJoomla)

To be release in a few hours from now
