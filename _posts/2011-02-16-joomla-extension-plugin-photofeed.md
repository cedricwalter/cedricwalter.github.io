[PhotoFEED](http://wiki.waltercedric.com/index.php?title=Photofeed_fo_Joomla) is a small content plugin for #Joomla! 1.5 that allow you to inline a set of images from your favorite online gallery: Smugmug, Flickr, Picasa or any RSS feed in any article.

You can [see it running here](http://demo-joomla-1.5.waltercedric.com/index.php?option=com_content&view=article&id=48:photofeed&catid=1:latest-news) in my Demo site.

### New version 1.5.0 is available!

[![photofeed.joomla.1.5_thumb3](/assets/images/2011/02/photofeed.joomla.1.5_thumb3_thumb.png "photofeed.joomla.1.5_thumb3")](/assets/images/2011/02/photofeed.joomla.1.5_thumb3.png)

# NEW**: use /media directory for resources   
# NEW**: online documentation in administrator panel   
# NEW**: plugin log data in debug mode at /logs/plg.photofeed.log.php you cant point a browser to it, use FTP/SCP/Plesk/Cpanel to read this file.
