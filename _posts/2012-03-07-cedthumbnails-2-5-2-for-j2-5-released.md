[CedThumbnails](https://waltercedric.com/new/?p=7429) has been updated to version 2.5.2 and contains 2 bug fixes and compatibility for both #Joomla 1.7 and #Joomla 2.5. For existing users the update will display in the **Extensions Manager** under **Updates**. If you do not have this currently installed, you can click the link below and install as normal via the Extensions Manager.

New ideas/features/bugs are always welcomed either through [forums](http://forums.waltercedric.com/) or by using [feedback](https://waltercedric.com/new/?p=7497)

## What’s new?

Update mod\_related\_items\_thumb to 2.5.2:

- **BUG**: item route was not working in #Joomla 2.5.0+
- **BUG**: module CSS class was not propagated to module view default.php

Full list of changes available in [GIT](http://git-scm.com/) commit [c6de1f0c70e834af2d95b4f91ecfbb0260f5a1e6](https://github.com/cedricwalter/cedthumbnails/commit/c6de1f0c70e834af2d95b4f91ecfbb0260f5a1e6)
