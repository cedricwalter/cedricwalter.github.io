[CedThumbnails](https://waltercedric.com/new/?p=7429) has been updated to version 2.6.0 and contains 4 new features and correct 1 bug for #Joomla 2.5. For existing users the update will display in the **Extensions Manager** under **Updates**. If you do not have this currently installed, you can click the link below and install like you are use to via the Extensions Manager.

# NEW**: All choices for layout styles (many missing in different modules),

# NEW**: Improved Module caching for Popular, Related and Latest Posts,

# NEW**: Improved detection of images for non XHTML posts content,

# NEW**: add Google+ social icon in backend,

# NEW**: Installer updated from 1.8 to 1.10,

# BUG**: Original (English) language file were containing some errors (<http://forums.waltercedric.com/index.php?topic=1399.0>)
