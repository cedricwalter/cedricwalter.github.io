> [Git](http://git-scm.com/) is a free &amp; open source, distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
> 
> Every [Git](http://git-scm.com/) clone is a full-fledged repository with complete history and full revision tracking capabilities, not dependent on network access or a central server. Branching and merging are fast and easy to do.

[Git](http://git-scm.com/) is an [entirely different beast](http://git.or.cz/course/svn.html) from Subversion, so it took me a bit to figure out exactly how to use it. Here is a collection of [Git](http://git-scm.com/) cheat sheet including version control best practices.
