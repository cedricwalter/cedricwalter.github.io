[RSSVoyage](http://rssvoyage.com) is a Flash based RSS reader, which use the mouse wheel only, just roll wheel to see news from up to 2 days ago. If only it would be available as a Firefox plugin…

[![rssvoyage](/assets/images/2009/02/rssvoyage_thumb.jpg)](/assets/images/2009/02/rssvoyage.jpg)

Add new RSS is done in the bottom bar (subscribe to my RSS by adding [http://www.waltercedric.com/index.php?format=feed&amp;type=rss](http://www.waltercedric.com/index.php?format=feed&type=rss) for example :-))
