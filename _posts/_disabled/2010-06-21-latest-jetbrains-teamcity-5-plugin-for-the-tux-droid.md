[Attitunes](http://www.kysoh.com/tux-droid/who-is-tux-droid/attitunes) *are*

> *robot animations you can create combining movement, sound, Text-To-Speech and fun. Make Tux Droid sing ‘Dancing in the rain’ or make him cry when you didn’t receive any new E-mail.*

The new version is now 0.0.2-SNAPSHOT, and I am able to support both [TeamCity 4.x](http://www.jetbrains.com/teamcity/) and [TeamCity 5.x](http://www.jetbrains.com/teamcity/)

Visit the site <http://tuxdroid.waltercedric.com/> to grab the latest version

This project is Open Source (GPL V3.0) and the source code is available at [Google Code](http://code.google.com/p/teamcitytuxdroidbuildplugin/), you can [join the project by contacting me](http://waltercedric.com/-contact-me.html)

You can read the [online documentation in my WIKI](http://wiki.waltercedric.com/index.php?title=Tux_Droid_Plugin_For_TeamCity_Continuous_BuildServer#Download)

The Teamcity builds (latest trunk) are also [available as artifacts in my public Teamcity server](http://teamcity.waltercedric.com/teamcity/viewType.html?tab=buildTypeStatusDiv&buildTypeId=bt2) (No login required).
