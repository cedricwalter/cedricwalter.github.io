I receive my new notebook yesterday, it replace my HP NX7000 (3 years old). The [HP NX 9440](http://h10010.www1.hp.com/wwpc/us/en/sm/WF25a/321957-64295-89315-321838-f1-1839859.html) is showing impressive performances, especially when running [eclipse ](http://www.eclipse.org)and multiple aplication in the backgroung.  
The specifications are as follow:

 3 GB (667 MHz) DDR2 SDRAM

 100 GB (7200 rpm) SATA  
 LightScribe DVD+/-RW with Double Layer  
 17.0-inch WUXGA wide viewing angle

 1 Type I/II PC Card slot  
 1 Integrated Smart Card Reader  
 1 7-in-1 Media ReaderBut there is somevery strange technical choice, like

- The position of all plugs, USB, VGA, network on both side which restrict mouse movements, a dubious decision!
- The speaker are in front, below the case (pointing 45° into the ground) : crazy
- On the back, you’ll find the battery, very thin and without any mention of capacity!
- The keyboard is one more time NOT standard (as on the HP NX7000), crazy layout of numpad.

On the plus side, the graphic card (GPU NVIDIA Quadro) is able to play full HD (1980 x 1080 or 1080p) in full screen without any glitches! Except than that, you can read numerous reviews:

- [Pc Pro](http://www.pcpro.co.uk/reviews/88744/hp-compaq-nw9440.html) 5 stars/6
- [NoteBookReview ](http://www.notebookreview.com/default.asp?newsID=3450&review=HP+nw9440)came to the same conclusions. This site has an extensive review.

Ive put an impressive list of open source on it, thus limiting the total cost of licence ownership, the only commercial tools are: Microsoft Office because a lot of documents are floating around which can only be open in M$ Office, Symantec Antivirus, Groove network. The rest is made of the best open source tools:

- [Gimp ](http://www.gimp.org)for manipulating 2D images
- [Inkscape ](http://www.inkscape.org)for vector graphics (SVG)
- [Videolan ](http://www.videolan.org)for playing all video content wihtout DRM hell
- [WinSCP ](http://winscp.net/)for connecting to server in a secure way,
- [CYGWIN ](http://www.cygwin.com)the only viable shell for developer
- [NVU ](http://www.nvu.com/)for limited HTML edition support
- [TrueCrypt ](http://www.truecrypt.com)for creating encrypted vault
- [Firefox ](http://www.firefox.com)for developing webapplications thanks to firefox extended plugins support
- [XNView ](http://www.xnview.com)as image viewer, conversion tool
- [Tight VNC](http://www.tightvnc.com/), [Emu48 emulator](http://privat.swol.de/ChristophGiesselink/), [JEDIT editor](http://www.jedit.org/)
