[JoomlaCode.org](http://joomlacode.org/) is a resource for developers to build and manage Open Source software projects centred around #Joomla! With #JoomlaCode.org we aim to help build Open Source tools and extensions for the #Joomla! user community by providing resources to help developers work together in an Open Source spirit.
