Some people are now using MXcomment 1.0.7 in #Joomla! 1.5 (Legacy mode is required).   
Unfortunately MXcomment code is only able to support securityimages 4.X

After 15 minutes, it is now supporting securityimages 5.X and 4.X at the same time since   
it detect in which version of #Joomla!, and which version of securityimages you are using.

Other than that, I’ve done nothing else.

[Original MXcomment version can be found here](http://extensions.joomla.org/component/option,com_mtree/task,viewlink/link_id,2621/Itemid,35/) while the modified version is available in   
my download section ([com\_MXcomment\_v1.0.7+ for securityimages 4.X and 5.X.zip](https://waltercedric.com/new/?p=215)) at   
the moment and as long as the author [Bernard Gilly](http://www.visualclinic.fr) do not review and use my changes in   
the standard version.
