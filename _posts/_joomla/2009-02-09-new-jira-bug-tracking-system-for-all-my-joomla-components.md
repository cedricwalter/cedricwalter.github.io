# I am very proud to announce that I got granted a JIRA and Bamboo Open Source license!

> ##### Extract form their site:   
> *Open Source*
> 
> *Atlassian supports and believes in the Open Source movement –* [*JIRA*](http://www.atlassian.com/software/jira/) *utilises a number of good Open Source components, and Atlassian developers are committers on a large number of Open Source projects.*
> 
> *To give back to the open source community (and hopefully improve the quality of those projects!),* [*JIRA*](http://www.atlassian.com/software/jira/) *is free for any Open Source project to use.*
> 
> *There are a few requirements for an Open Source license, the main ones being:*
> 
> - *Established code base*
> - *Publicly available project website*
> - *Using an approved open source license*
> - *Your* [*JIRA*](http://www.atlassian.com/software/jira/) *instance will be publicly accessible*
