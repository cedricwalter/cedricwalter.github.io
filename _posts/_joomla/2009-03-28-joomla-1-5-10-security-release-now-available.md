From [Joomla.org](http://www.joomla.org/announcements/release-news/5231-joomla-1510-security-release-now-available.html)

*The #Joomla Project announces the immediate availability of #Joomla 1.5.10 \[Wohmamni\]. This is a security release and users are strongly encouraged to upgrade immediately.   
This release contains 66 bug fixes, one low-level security fix, and one moderate-level security fix. It has been 11 weeks since* [*Joomla 1.5.9 was released*](http://www.joomla.org/announcements/release-news/5226-joomla-159-security-release-now-available.html) *on January 10, 2009. The* [*Development Working Group’s*](http://docs.joomla.org/Development_Working_Group) *goal is to continue to provide regular, frequent updates to the #Joomla community.*

# Statistics for the 1.5.10 release period:

- Joomla 1.5.10 contains: 
    - 68 issues fixed in SVN
    - 281 commits
- Tracker activity resulted in a net decrease of 8 active issues: 
    - 176 new reports
    - 133 closed
    - 68 fixed in SVN
- At the time the 1.5.10 release was packaged, the tracker had 95 active issues: 
    - 44 open
    - 40 confirmed
    - 11 pending
