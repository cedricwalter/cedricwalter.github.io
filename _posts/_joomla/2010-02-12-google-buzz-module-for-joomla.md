Here is my Google Buzz module for #Joomla! [based on Google Buzz ER](http://wordpress.org/extend/plugins/google-buzz-er/screenshots/) GNU/GPL code of [Ivan Lazarevic](http://www.workshop.rs). Note that I did convert it into a #Joomla! module in less than 15 minutes!

![google.buzz.joomla.module](/assets/images/2010/02/google.buzz_.joomla.module_3.png "google.buzz.joomla.module")

- Native module for #Joomla! 1.5
- Use latest PHP5
- Caching
- CSS layout and style
- Dead simple code that use [www.simplePie.org](http://www.simplePie.org) RSS parser
