I did convert some modules to be native under [Joomla!](http://www.joomla.org/) 1.6, they now use the update manager, and click once update mechanism of [Joomla!](http://www.joomla.org/) 1.6

### Smugmug Badge Picture module

![Smumug.badge.joomla.02](/assets/images/2010/08/Smumug.badge_.joomla.02.png "Smumug.badge.joomla.02")A small module, highly customizable to display a badge of your [Smugmug](http://www.smugmug.com/) galleries (up to 4 galleries are randomly supported).

#### Features

- Native [Joomla!](http://www.joomla.org/) 1.6 module,
- Code produced is XHTML 1.0 Strict,
- Tested in the following browser Firefox, Opera, Chrome, IE 8
- [Approved at #Joomla Extension JED in category Photo Channels,](http://extensions.joomla.org/extensions/photos-a-images/photo-channels/9123)
- GPL v3.0,
- Use Adobe Flash 10
- Non commercial.

#### [Download mod\_smugmugbadge\_1.6.0.zip](https://waltercedric.com/new/?p=555) – [Wiki](http://wiki.waltercedric.com/index.php?title=Smugmug_Badge)

### Smugmug Random Picture module

![Smumug.random.joomla](/assets/images/2010/08/Smumug.random.joomla.png "Smumug.random.joomla") A small module, highly customizable to display picture randomly from your [Smugmug](http://www.smugmug.com/) gallery.This module displays [Smugmug](http://www.smugmug.com/) photos using lightbox, lightbox2 or shadowbox.

Features

- Native #Joomla! 1.6 module,
- Code produced is XHTML 1.0 Strict,
- Tested in the following browser Firefox, Opera, Chrome, IE 8
- [Approved at #Joomla Extension JED in category Photo Channels,](http://extensions.joomla.org/extensions/photos-a-images/photo-channels/8950)
- GPL v3.0,
- Non commercial.
