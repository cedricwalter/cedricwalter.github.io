 This small plugin add automatically to any articles a set of social icons that let your reader increase your social ranking. It support

- [Twitter](http://twitter.com/)
- [Facebook](http://www.facebook.com/)
- [Google Buzz](http://www.google.com/buzz)
- [Digg](http://digg.com/)
- [LinkedIn](http://www.linkedin.com/) There is 2 different branches of this plugins depending on your version of #Joomla!
- [Download version 1.6.x for #Joomla! 1.6.x](https://waltercedric.com/new/?p=659), a running demo of this extension can be seen at <http://demo-joomla-1.6.waltercedric.com/>
- [Download version 1.5.x for #Joomla! 1.5.x](https://waltercedric.com/new/?p=642), a running demo of this extension can be seen at <http://demo-joomla-1.5.waltercedric.com/> or you can see it in action on this site ([I eat my own dog food](http://en.wikipedia.org/wiki/Eating_your_own_dog_food))  [![joomla_socialsharing_horizontal_left](/assets/images/2011/04/joomla_socialsharing_horizontal_left_thumb.png "joomla_socialsharing_horizontal_left")](/assets/images/2011/04/joomla_socialsharing_horizontal_left.png) [![joomla_socialsharing_left](/assets/images/2011/04/joomla_socialsharing_left_thumb.png "joomla_socialsharing_left")](/assets/images/2011/04/joomla_socialsharing_left.png)
    
    ##  Features
    
    
    - Output is CSS driven, 6 preset styles
    - Toolbar can be added before or after your article
    - It is possible to include exclude sections and category for displaying the toolbar
    - Inclusion of #JavaScript done using asynchronous techniques for more performances
    - Less than 220 lines of PHP code!
    - Support for SEF friendly URLs
    - Code produced is XHTML 1.0 Strict,
    - License under GNU/GPL v3
    - Tested in the following browser Firefox, Opera, Chrome, IE 8
