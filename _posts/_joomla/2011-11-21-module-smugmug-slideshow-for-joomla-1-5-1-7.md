This module is a front end for the myriad of parameter that can set for the [SmugMug](http://www.smugmug.com/) flash
applet.A great visual and a lot of features are built into this widget.

[![Module_Smugmug_Slideshow_for_Joomla_1.5_1.7](/assets/images/2011/11/Module_Smugmug_Slideshow_for_Joomla_1.5_1.7_thumb.png "Module_Smugmug_Slideshow_for_Joomla_1.5_1.7")](/assets/images/2011/11/Module_Smugmug_Slideshow_for_Joomla_1.5_1.7.png)

Flash 10 widget – Code produced is XHTML 1.0 Strict – Tested in the following browser Firefox, Opera, Chrome, IE 8

Note: [SmugMug](http://www.smugmug.com/) is a paid digital photo sharing website, founded by Chris and Don MacAskill in
2002.

[Demo](http://demo-joomla-1.7.waltercedric.com/)

##### We’d love to turn your access back on, so please [email AnchorFree](mailto:support@anchorfree.com) and ask them to help us.
