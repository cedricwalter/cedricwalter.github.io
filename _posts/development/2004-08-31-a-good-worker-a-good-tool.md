# A good worker. a good tool!
- Use only [Eclipse](http://www.eclipse.org/) and try to buy [codepro](www.instantiations.com) or found a free plugin that create java pattern templates and has refactoring capabilities..
- Use only Borland if you need to do a RAD (Rapid applications development) applications
- And for the text editor or xml or xsl or whatever, use only [JEDIT](www.jedit.org), it is free and for sure the best editor, many of my colleagues are still fighting, convince that ultraedit or notepad or mpad are a lot better. You are clever only if you accept sometimes you are not right….
- TCPviewer and Process viewer, free tools that can be download at [www.winternals.com](http://www.winternals.com/)
