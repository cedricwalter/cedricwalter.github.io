- *[Azureus](http://azureus.sourceforge.net/) is a java bittorrent client. It provides a bittorrent protocol
  implementation using java language. [Azureus](http://azureus.sourceforge.net/) offers multiple torrent downloads,
  queuing/priority systems (on torrents and files), start/stop seeding options and instant access to numerous pieces of
  information about your torrents.  
  [![](http://azureus.sourceforge.net/img/azureus.jpg)](http://azureus.sourceforge.net/)*
- *[ComicViewer](http://lokoen.org/comicviewer/index.php) is a browser for online comics. Using a simple interface you
