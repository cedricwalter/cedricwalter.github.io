The[ Web Tools Platform ](http://www.eclipse.org/webtools/)(WTP) project’s mission is to provide a common framework for
building Web tools for [Eclipse ](http://www.eclipse.org)and to provide both standard and #Java Web tools that build
upon the common framework.
