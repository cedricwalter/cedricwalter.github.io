[IBM Alphaworks](www.alphaworks.ibm.com) have release a library for supporting the IETF SSH-2 protocol aka[ SSH (WikiPedia)](http://en.wikipedia.org/wiki/SSH)

[IBM Secure Shell Library for #Java](http://www.alphaworks.ibm.com/tech/sshlite?open&S_TACT=105AGX59&S_CMP=GR) is a lightweight implementation of the IETF SSH-2 protocol. The library currently implements only the basic SSH features such as password log-in and command execution. Advanced features such as tunning and X-forwarding are currently not supported.
