I compiled here a list of the major [Apache #Maven](http://maven.apache.org/) repositories ([read an intro to repositories](http://maven.apache.org/

- <http://maven.atlassian.com/repository/public> The Atlassian public repo
- <http://mvn.carbonfive.com/nexus/content/groups/public>
- <http://repository.codehaus.org> Codehaus releases repository
- <http://google-maven-repository.googlecode.com/svn/repository> Google’s open source releases
- <http://gradle.artifactoryonline.com/gradle/libs> Gradle libraries (internal and external)
- <http://download.java.net/maven/2> java.net #Maven2 Format
- <http://download.java.net/maven/1> java.net #Maven1 format
- <http://repository.apache.org> The main snapshot and release repositories, all release are synced to [Central](http://repo1.maven.org/maven2)
- <http://repository.jboss.org/maven2> JBoss Release Repository
- <http://snapshots.jboss.org/maven2> JBoss Snapshot Repository
- <http://repo.jfrog.org/artifactory/libs-releases> JFrog libs releases
- <http://repo.terracotta.org/maven2> Terracotta releases
- <http://repo.jfrog.org/artifactory/plugins-releases-local> JFrog plugins releases
- <http://repo.jfrog.org/artifactory/repo1> The #Maven Repo1 Cache proxy on repo.jfrog.org
- <http://nexus.openqa.org/content/repositories/releases> OpenQA (Selenium) release
- <http://download.oracle.com/maven> Oracle releases
- <http://repo1.maven.org/maven2> Central #maven repository
- <http://scala-tools.org/repo> Scala-Tools.org releases
- <http://maven.springframework.org/milestone> SpringSource bundles in milestone version
- <http://maven.springframework.org/release> SpringSource releases
- <http://maven.springframework.org/snapshot> SpringSource bundles snapshots
- <http://repository.springsource.com/maven/bundles/release> SpringSource EBR repositository
- [http://el4.elca-services.ch/el4j/maven2repository](http://el4.elca-services.ch/el4j/maven2repository "http://el4.elca-services.ch/el4j/maven2repository") ELCA (<http://www.elca.ch> ) #Maven plugins
- [http://www.eviware.com/repository/maven2/](http://www.eviware.com/repository/maven2/ "http://www.eviware.com/repository/maven2/") SOAPUI #Maven plugins

And of course mine at <http://maven.waltercedric.com>
