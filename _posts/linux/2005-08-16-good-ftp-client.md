If You are in the business for searching a good FTP client to maintain Your homepage under linux and are already deseperate with KBEAR stability (default KDE ftp client). then You are ready for trying one of these:

- <font face="Lucida,Verdana,Helvetica,Arial">[ gFTP](http://gftp.seul.org/) is a free multithreaded file transfer client for \*NIX based machines. You can install it using YaST. It is a lot more user friendly (if You’re working with windows ftp client) and a loooot faster than kbear…I only using this one now.   
    
- <font>[GUI FTP client for X Window System](http://www.wxftp.seul.org/) (looks like WFTP old windows ftp client) but it is also working nicely.

YOu can visit the screenshots section for both of these tool before deciding which suit more for the job.
